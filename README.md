# Air Quality (AQI) Forecasting & Pattern Analysis — Delhi

This project analyzes Delhi air quality (AQI) using historical data (2015–2023).
It contains EDA, time-series forecasting, clustering, and seasonal insights.

## Folder structure
- data/        → dataset CSV
- notebooks/   → Jupyter notebook (analysis)
- src/         → scripts (optional)
- results/     → plots and images

## How to run
1. Open `notebooks/air_quality_analysis.ipynb` in Colab or Jupyter.
2. Install dependencies (pandas, numpy, matplotlib, seaborn, scikit-learn).
3. Run notebook cells from top to bottom.
